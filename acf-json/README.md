# acf-json

Dossier de synchronisation automatique des field groups ACF.

ACF sauvegarde ici en JSON les field groups créés/modifiés depuis l'admin WP.
Ces fichiers sont versionnés dans le repo — ils sont la source de vérité.

Pour importer les field groups sur une nouvelle install :
1. Activer le thème
2. Admin → Custom Fields → Field Groups
3. Cliquer "Sync" si un bandeau apparaît
